# LEICAM Wine Catalog

Sistema de catálogo digital de vinhos para restaurantes.

## Tecnologias

### Backend

* Node.js
* Express
* MySQL
* Zod
* Multer

### Recursos

* Cadastro de vinhos
* Cadastro de uvas
* Cartas de vinho
* QR Code para acesso público
* Estatísticas de acesso
* Upload de imagens

## Estrutura

* Restaurants
* Wine Styles
* Grapes
* Wines
* Wine Lists
* Access Logs
